# Instrução para rodar:
## Na pasta backend:
node api-gateway.js
node controle.js
node logging.js

## Na pasta frontend:
npx expo start

## Embarcado (esp32):
Adicionar o código na placa via IDE (Arduino IDE) e deixar rodando.



# Requirements:
## Para o backend:
npm install express sqlite3 body-parser cors express-http-proxy morgan
